# game-valuation

> YY 游戏账号估值 Skill for Claude Code（源码版）

在 Claude Code 对话中直接查询游戏账号估值价格，支持王者荣耀、和平精英、三角洲行动三款游戏。

**无需登录或配置凭据**，开箱即用。API 签名密钥已内嵌在脚本中。

---

## 功能

- 查询游戏账号的预估价格、价格区间、超越用户比例
- 支持王者荣耀（直接出结果）、和平精英/三角洲行动（扫码验证后出结果）
- 展示账号核心数据：英雄数量、皮肤总数、段位、贵族等级等
- 自动处理扫码验证流程（二维码生成、自动打开、自动清理）

## 支持游戏

| 游戏 | 需要扫码 | 必填属性 |
|------|---------|---------|
| 王者荣耀 | 否 | 区服、营地ID、实名情况、防沉迷 |
| 和平精英 | 是（base64 二维码） | 区服、实名情况、防沉迷 |
| 三角洲行动 | 是（微信 URL 二维码） | 登录方式、实名情况 |

## 依赖

- **bash** — 脚本运行环境
- **python3** — JSON 解析、MD5 签名、base64 解码、报告格式化
- **curl** — HTTP 请求

```

→ 询问区服、营地ID、实名情况、防沉迷信息
→ 提交估值，直接返回结果

用户：和平精英账号多少钱
→ 询问区服、实名情况、防沉迷信息
→ 提交估值，生成二维码
→ 扫码验证后返回估值结果
```
